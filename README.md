# FIAP Bank ATM

> Simulador de terminal de autoatendimento bancário rodando via console, desenvolvido em Java.

---

## 📋 Sobre o Projeto

Projeto acadêmico desenvolvido para a disciplina de **Domain Driven Design Java**, simulando o fluxo completo de um caixa eletrônico físico, do cadastro de senha até as operações bancárias.

---

## ⚙️ Funcionalidades

| Funcionalidade | Descrição |
|---|---|
| 👤 Cadastro | Nome do cliente com boas-vindas personalizados |
| 🔐 Senha Forte | Validação com regex mínimo 8 caracteres, número, maiúscula e especial |
| 🚫 Bloqueio | Acesso bloqueado após 3 tentativas incorretas |
| 💰 Saldo | Consulta do saldo formatado |
| 💳 Depósito | Com validação de valor positivo |
| 💸 Saque | Com validação de saldo suficiente |
